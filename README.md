# Collaborative Drawing App 🎨✍️  

A real-time collaborative drawing application inspired by Excalidraw, built using **Next.js**. Collaborate seamlessly with others to create, edit, and share drawings in real time!  

## 🚀 Features  

- **Real-Time Collaboration**: Connect and draw with multiple users simultaneously using WebSockets.  
- **Intuitive Drawing Tools**: Freehand drawing, shapes, text, and more.  
- **Dynamic Canvas**: Responsive canvas that adjusts to window size changes.  
- **User-Friendly Interface**: Minimalistic and easy-to-use UI for smooth collaboration.  
- **Modern Tech Stack**: Built with Next.js for performance and scalability.
